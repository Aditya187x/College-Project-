# Secure Message Encryption

This Python project allows users to write a message and generate a security key.  
On another device, entering the correct key will reveal the original message.  
If an incorrect key is entered, the recipient will receive an encrypted message instead.

## Features
- Encrypt messages with a unique security key.
- Decrypt messages using the correct key.
- Prevents access to the original message with an incorrect key.

## Installation
1. Clone this repository:  
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```
2. Install dependencies (if any):  
   ```bash
   pip install -r requirements.txt
   ```
3. Run the script:  
   ```bash
   python main.py
   ```

## Usage
- Enter a message to encrypt.
- Save the generated security key.
- Use the key on another device to decrypt the message.
